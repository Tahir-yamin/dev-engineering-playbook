## Skill #1: OpenCode & Independent Agents

### OpenCode (`anomalyco/opencode`)
The open-source coding agent alternative to Claude Code.
- **Modes**:
    - `build`: Full access dev agent.
    - `plan`: Read-only exploration agent.
- **Usage**: Good for local execution without vendor lock-in. 
- **Install**: `brew install opencode`

## Skill #2: Agentic RAG Patterns

Advanced Retrieval-Augmented Generation patterns for 2026.

- **Corrective RAG (CRAG)**: Evaluates retrieved documents and seeks more if quality is low.
- **Agentic RAG with Reasoning**: Adds a reasoning layer before synthesis.
- **Hybrid Search**: Combines keyword (BM25) and vector search for best recall.
- **Vision RAG**: Incorporating multimodal inputs into retrieval.
