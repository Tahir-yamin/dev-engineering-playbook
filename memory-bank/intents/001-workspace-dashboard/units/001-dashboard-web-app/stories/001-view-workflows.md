# Story: View Workflows

**ID**: `001-view-workflows`
**Role**: Developer
**Goal**: I want to see a list of all available workflows so I know what automation commands I can run.

## Acceptance Criteria
1.  [ ] Navigating to `/workflows` displays a list.
2.  [ ] The list is populated from files in `.agent/workflows/`.
3.  [ ] Each item shows the filename (e.g., `specsmd-inception-agent.md`).
4.  [ ] Clicking an item shows the full markdown content of the file.
