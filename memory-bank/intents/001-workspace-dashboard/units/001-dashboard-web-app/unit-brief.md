# Unit Brief: Dashboard Web App

**ID**: `001-dashboard-web-app`
**Intent**: `001-workspace-dashboard`

## 1. Scope
This unit encompasses the entire Next.js application that serves as the dashboard.

## 2. Key Components
- **Layout**: Main navigation frame.
- **Workflow List Page**: Renders content from `.agent/workflows/`.
- **Rules List Page**: Renders content from `.agent/rules/`.
- **File Reader Utility**: Server-side logic to safely read markdown files.

## 3. Technology
- Next.js (App Router)
- React Server Components
- Tailwind CSS
